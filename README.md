---

## 🧠 About the Project

MBGastroStock is a stock management tool designed to simplify inventory control in gastronomy environments.

It helps track products, update stock levels quickly, and reduce manual errors in daily operations.

This project is currently in development and aimed for future commercialization.

---

## ✨ Features

- Manage products and inventory
- Update stock in real time
- Simple and intuitive interface
- Designed for fast kitchen workflows
- Lightweight and efficient
