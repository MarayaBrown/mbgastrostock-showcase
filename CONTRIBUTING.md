# Contributing

This is a **portfolio showcase project** for educational purposes only.

## Feedback & Discussions

We welcome feedback on the design, functionality, and implementation:

- **Issues** — Report bugs or suggest improvements via [GitHub Issues](https://github.com/mariabrownlejarraga/showcase-mbgastrostock/issues)
- **Discussions** — Share ideas and ask questions via [GitHub Discussions](https://github.com/mariabrownlejarraga/showcase-mbgastrostock/discussions)

## Pull Requests

**Pull requests are not accepted** to this repository. This is a read-only showcase of work.

However, you're welcome to:
- Fork the repository for your own learning
- Use the patterns and architecture as inspiration for your own projects
- Reference the code in your portfolio or discussions
- Study the implementation approach
