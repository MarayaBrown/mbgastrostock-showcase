# Contributing

This is a **portfolio showcase project**, not open source.

## Feedback & Discussions

We welcome feedback on the design, functionality, and implementation:

- **Issues** — Report bugs or suggest improvements via [GitHub Issues](https://github.com/mariabrownlejarraga/showcase-mbgastrostock/issues)
- **Discussions** — Share ideas and ask questions via [GitHub Discussions](https://github.com/mariabrownlejarraga/showcase-mbgastrostock/discussions)

## Pull Requests

**Pull requests cannot be merged** to this showcase repository. However:

- You're welcome to fork the repository and adapt it for your own projects
- Use it as inspiration for your own applications
- Build on the patterns and approach for your own inventory management system
